Private fix

  • fix(key: string | number, parent: any | Array<any>, schema: JSONSchema): void

  • Visits all parts of record recursivly, along with the subschema of the part and apply required fixes.

    NOTE: the reason that parent and key are passed instead of the direct value is to be able do some operations that needs the parent such as delete.

    TODO: add special case for arrays because fixes are the same for all elements.

      parent of the field/element

    • schema: JSONSchema

      schema of visited field/element

    Returns void
