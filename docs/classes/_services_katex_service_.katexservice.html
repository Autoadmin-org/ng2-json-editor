renderMathInText

  • renderMathInText(text: string, el: HTMLElement): void

  • Receives text that can contain LaTeX formulas. The formulas will be identified using this.delimiters and rendered inside the passed HTMLElement

    NOTE: The HTMLElement is expected to have a single child and this will be replaced by the HTML with formatted LaTeX

    Parameters

    • text: string

      text to be formatted with LaTeX

    • el: HTMLElement

      the HTMLElement where the LaTeX should be rendered

    Returns void
